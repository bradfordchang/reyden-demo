Methodology. Scenario SQL is pulled live from the selected dashboard's Lakeview datasets with default parameter values applied — the same queries the dashboard runs to hydrate its widgets. All of a run's queries fire concurrently — the same burst a dashboard sends when it loads. Every statement carries a unique tag to defeat result caches and runs through the SQL Statement Execution API on your identity (on-behalf-of authorization); bars show wall-clock time to results-ready, medians across runs. Both lanes warm up before the start gun so cold starts never skew a lap.
