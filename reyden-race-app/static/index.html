Methodology. Scenario SQL is generated from the live Lakeview dashboard datasets (a Unity Catalog metric view over liquid-clustered copies of system.billing tables) and shaped exactly like widget queries — page defaults plus filter/parameter variants. Every statement carries a unique tag to defeat result caches and runs through the SQL Statement Execution API; bars show wall-clock time to results-ready, medians across runs. Both lanes warm up before the start gun so cold starts never skew a lap.
